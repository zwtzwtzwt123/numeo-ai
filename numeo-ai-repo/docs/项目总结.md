# Numeo AI 通用计算平台 — 项目状态总结

## 一、项目概况

**定位：** AI 通用计算平台，覆盖 7 大领域，620+ 计算功能，支持中英文混合输入。

**技术栈：**
- 后端：Cloudflare Workers（单文件，约 1 万行）
- 前端：React + TypeScript + Tailwind CSS + Vite
- AI：Cloudflare Workers AI (@cf/meta/llama-3-8b-instruct)

### 后端关键架构
- `MODULE_REGISTRY` — 模块注册表
- `classifyIntent` — AI 意图分类（调用 LLaMA）
- `fallbackClassify` — 后备关键词分类
- `routeToModule` — 模块路由器（含大量转发逻辑）
- 每个函数内部有 `getK` 参数提取函数

### 已知待修复问题
1. `solve x^2+2x+1=0`、`derivative`、`极限` — AI 模型超时/返回空
2. 电磁学与电气之间转发需注意循环
3. 部分科学计数法参数提取需要命名匹配
